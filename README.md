# 📡 RINEX Downloader

Автоматический загрузчик RINEX-файлов с сети базовых станций [EFT-CORS](https://bp.eft-cors.ru) для станций Москвы и Московской области.

---

## О проекте

**RINEX** (Receiver Independent Exchange Format) — стандартный формат хранения данных спутниковых наблюдений ГНСС (GPS, ГЛОНАСС и др.). Такие файлы используются в геодезии, картографии, научных исследованиях и при постобработке координат.

Сайт **bp.eft-cors.ru** предоставляет доступ к архиву данных со своей сети базовых станций, однако скачивание файлов вручную через браузер — процесс долгий и монотонный: нужно выбрать станцию, задать временной диапазон, нажать «Подождать ссылку», дождаться генерации файла, скачать — и повторить для каждой станции.

Этот скрипт **автоматизирует весь процесс**: авторизуется на сайте, формирует запросы, ожидает готовности файлов через SSE-поток и скачивает их в структурированные папки — без участия пользователя.

## Возможности

- 🔐 Автоматическая авторизация на сайте
- 📋 Встроенный список всех 19 станций МСК и МО с кодами
- 🕐 Интерактивный выбор временных диапазонов и станций перед запуском
- 📁 Сортировка файлов по папкам в зависимости от диапазона измерений (`12-14`, `12-16`, `12-18`)
- ⏱ Автоматический кулдаун 62 минуты при достижении лимита сайта (10 файлов/час) с обратным отсчётом в терминале
- 🔁 Продолжение скачивания с того же места после кулдауна — без потери прогресса
- 🚨 Определение кулдауна при старте — если лимит был исчерпан до запуска программы
- 📝 Подробное логирование в файл `rinex_parser.log`

## Структура файлов после скачивания

```
📁 rinex_files\
    ├── 📁 12-14\        ← измерения 12:00 – 14:00 МСК
    │     ├── VOSK_020226.0900_020226.1059_xxxxx.26o
    │     ├── DMTR_020226.0900_020226.1059_xxxxx.26o
    │     └── ...
    ├── 📁 12-16\        ← измерения 12:00 – 16:00 МСК
    │     └── ...
    └── 📁 12-18\        ← измерения 12:00 – 18:00 МСК
          └── ...
```

> Время в именах файлов и внутри RINEX-заголовка указано в **UTC** (МСК − 3 ч) — это стандарт формата RINEX 2.11.

## Быстрый старт

```bash

pip install requests beautifulsoup4 lxml

```

После запуска программа интерактивно спросит:
1. Какие временные диапазоны скачивать
2. Какие станции выбрать

Затем запустится автономное скачивание — с автоматическими паузами и кулдаунами.

## Технические детали

- Для получения файла используется эндпоинт `/json/get-rinex` с параметрами `rinex[bs]`, `rinex[measure_start]`, `rinex[measure_end]`, `rinex[timezone]=3`, `rinex[version]=2.11`, `rinex[frequency]=10`
- Готовность файла отслеживается через **Server-Sent Events** (`text/event-stream`) — сервер присылает два события: `{"converting": 0}` и затем `{"file_name": "..."}` 
- Файл скачивается напрямую по пути `/rinex/{file_name}`

## Ограничения

- Сайт bp.eft-cors.ru ограничивает скачивание **10 файлами в час** — программа учитывает это автоматически
- Для работы необходим активный аккаунт на bp.eft-cors.ru
- Некоторые станции могут временно не иметь данных за запрошенный период (сервер вернёт `{"result": false, "reason": "no files"}`)
